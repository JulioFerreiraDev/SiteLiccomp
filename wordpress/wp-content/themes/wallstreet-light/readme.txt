A Business theme which is ideal for creating a corporate / business website. Wallstreet Light theme is a child theme of Wallstreet theme and it is the lite color variation. Those who need the white background based theme they can use this child  theme and if you want dark background based then directly use the parent theme. Wallstreet Light theme comes with Spanish translation. Other translations / locals yet to come.
